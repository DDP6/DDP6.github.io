About Me!

I made the site with this basic black & white color scheme because it looks far more professional then then other flat colors. Creating this site was not an easy thing to do. The parts of it weren't hard, but putting them all together in a way that worked and didn't look ugly was pretty hard. Most of the time whatever was wrong was a simple spelling mistake, or a word that should be capitalized but isn't. The navigation bar gave me far more trouble then the rest of the site, I had to make some changes, such as making the current page not appear on the navbar. The reason the navbar was so hard is because I still don't fully understand the code behind it, as its bootstrap and I don't have the time to look at all of bootstrap's code bit by bit.
